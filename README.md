# JS Proficiency Passport AI — Professional Cohort Edition

This package contains the professional static deployment edition of JS Proficiency Passport AI for English proficiency diagnosis, structured practice, ITP-style secure testing, score reporting, certificate generation, certificate registry, QR-style verification, proctoring review, and institutional analytics.

## Deployment
Upload the extracted files directly to the root of the GitHub repository `JSproficiency`:

- `index.html`
- `styles.css`
- `app.js`
- `manifest.json`
- `README.md`

Then activate GitHub Pages from `Settings -> Pages -> Deploy from a branch -> main -> /root`.
